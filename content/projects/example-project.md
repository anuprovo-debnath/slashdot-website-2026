The flagship digital presence for Slashdot, the coding club of IISER Kolkata. 
This project features a high-fidelity design system, responsive card grids, 
and a state-of-the-art theme engine utilizing the Browser View Transitions API.

Built with performance and aesthetics in mind, it serves as the central hub 
for blog posts, project showcases, and member contributions.
